Introduction

Mapped back the haplotypes to the children and respective infection episodes. The overall trend of the top 4 haplotypes i.e E-KSNG-L, Q-KSNG-F, Q-KSNG-L and Q-KNNG-L was defined using local weighted regression model to fit smooth curve.

Haplotype distribution

library("tidyverse")

hap_den<- readxl::read_xlsx("data/Haplotype Distribution.xlsx", col_names = TRUE)
hap_den<-rbind(hap_den)
den<-ggplot (hap_den,aes(y= Frequency, x=Episodes)) +
  stat_smooth(aes(group= Haplotype, color= Haplotype, fill=Haplotype), method="loess", size=1, formula="y ~ log(x)", se=FALSE) +
  geom_point(aes(colour=Haplotype))+facet_grid( Haplotype ~ . )+
  labs(x="Infection episodes", y = "Frequency")+
   theme_classic(base_size = 10) +
  #theme(aspect.ratio=1/2.5)
theme(text = element_text(size = 20))
den

pdf("figures/haplotype_distribution.pdf", width = 30, height = 16, pointsize=30)#save as pdf file
den
dev.off()
